# Clipd

一款常驻菜单栏的 macOS 剪贴板历史管理工具(类 Paste)。原生 Swift 6 + SwiftUI + AppKit。

## 技术栈

- **平台**:macOS 14+(Apple Silicon),Swift 6 严格并发
- **持久化**:SwiftData(`@ModelActor` 隔离写入)+ 文件落盘(图片/大内容)
- **全局热键**:[KeyboardShortcuts](https://github.com/sindresorhus/KeyboardShortcuts)(默认 ⌘⇧V)
- **键码映射**:[Sauce](https://github.com/Clipy/Sauce)(非 QWERTY 布局正确粘贴)
- **工程生成**:xcodegen(源真相 `project.yml`)

## 构建 / 运行 / 测试

```bash
# 生成 Xcode 工程
xcodegen generate

# 构建
xcodebuild -project Clipd.xcodeproj -scheme Clipd -configuration Debug build

# 跑单元测试(32 个)
xcodebuild -project Clipd.xcodeproj -scheme Clipd -destination 'platform=macOS' test

# 运行:用 Xcode 打开 Clipd.xcodeproj 后 Run,或 open 构建产物
open ~/Library/Developer/Xcode/DerivedData/Clipd-*/Build/Products/Debug/Clipd.app
```

> 工程文件 `Clipd.xcodeproj` 由 xcodegen 生成,不手动编辑;改依赖/结构请改 `project.yml` 后重跑 `xcodegen generate`。

## 首次使用

1. 启动后菜单栏出现剪贴板图标(无 Dock 图标)。
2. 复制任意文本/图片 → 自动进入历史。
3. 按 **⌘⇧V** 唤起搜索面板;↑↓ 选择、⏎ 粘贴回当前 App、esc 关闭。
4. **自动粘贴需授予辅助功能权限**:系统设置 → 隐私与安全性 → 辅助功能,勾选 Clipd(授权后建议重启 app)。未授权时降级为"只复制",需手动 ⌘V。

## 隐私

- 不存储密码:命中 `org.nspasteboard.ConcealedType / TransientType / AutoGeneratedType` 标记的内容直接跳过。
- 自身写回带私有 marker,避免历史里出现"镜像副本"。

## 架构

```
App/        @main 入口、AppDelegate、AppCoordinator(组合根)
ClipdCore/
  Models/        Sendable 领域模型
  Persistence/   ClipItemEntity(@Model)、@ModelActor 仓储
  Storage/       BlobStore(文件落盘)、AppPaths
  Clipboard/     监听、隐私过滤、捕获编排、缩略图、裁剪
  Paste/         写回、键码映射、粘贴服务
  Hotkeys/       全局热键
  Permissions/   辅助功能权限
  Panel/         NSPanel + 状态项
  Features/History/  SwiftUI 面板视图 + Store
ClipdTests/   单元测试(隐私/捕获/去重/防回环/裁剪/仓储/缩略图/blob)
```
